# Zoo Binary Repository

Welcome to the official **ZOO** repository for the **Root Core** ecosystem.

## Overview
**Root Core** is an operating system built on top of the **Root Core Custom** kernel. In this environment, **ZOO** refers to the native binary format and application type. This repository functions as the central storage, distribution, and package hub for binary programs managed by the **Zoo Package Manager** within the OS.

## Usage & Contributions
This repository serves primarily as a static binary registry/storage. For regular users, there isn't much to do here directly other than fetch packages via the manager.

However, if you are a developer looking to expand the Root Core ecosystem, we welcome your creations! This repository is driven by community contributions. Please review our [CONTRIBUTING.md](CONTRIBUTING.md) guide to learn how to submit your applications, libraries, or drivers.
